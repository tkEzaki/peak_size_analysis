# Description
This python package provides functions to evaluate two-dimensional patterns of presynamptic activities (Yamagata et.al. 2021 (to be updated)).

# Requiements
This package assumes python 3.6 or later.
This package requires the following package:
-NumPy (https://numpy.org/)
